## 🤖 No que estou trabalhando agora

Meu foco hoje vai bem além da plataforma Salesforce. Estou investindo pesado em **engenharia de IA agêntica** — desenhando squads autônomos e multi-agentes capazes de assumir o ciclo completo de entrega, desde uma ideia ainda crua em uma issue do Jira até um merge request pronto para revisão humana.

Na prática, isso se desdobra em algumas frentes conectadas:

- **🎫 Automação de tarefas no Jira** — agentes de IA que acompanham os tickets que entram, interpretam os requisitos, buscam o contexto certo no código-fonte, documentação e issues anteriores, e então fazem triagem, refinamento, estimativa e decomposição antes mesmo de qualquer humano tocar a tarefa. O objetivo é eliminar a maior parte do overhead de grooming e começar toda demanda já com um plano claro e acionável.

- **🛠️ Pipelines de CI/CD com IA integrada** — incorporando passos guiados por modelos de IA diretamente em fluxos do Jenkins, GitLab e Azure DevOps, de forma que geração de testes, análise estática, code review, varredura de segurança, release notes e validação de deploy aconteçam automaticamente como parte do build. A pipeline passa a ser um revisor ativo, e não apenas um executor passivo.

- **🤝 Entrega autônoma ponta a ponta** — a visão maior: um agente pega uma issue do Jira, planeja a implementação, escreve o código, roda e itera os testes, valida contra os critérios de aceite e abre um merge request no cliente Git com um diff limpo e um resumo escrito, pronto para um humano aprovar. Na prática, um colega de time movido a IA seguindo o mesmo workflow que os engenheiros já usam.

- **🧠 Rigor de engenharia aplicado a prompts** — versionando prompts, construindo harnesses de avaliação, definindo guardrails e observabilidade para o comportamento dos agentes, e tratando toda a camada de orquestração como software de produção. A mesma disciplina que aplico a metadata, Apex e DevOps Salesforce, agora aplicada a workflows movidos por LLMs.

A aposta maior é que entregar software vai se tornar tão sobre **supervisionar agentes** quanto sobre escrever código você mesmo — e a entrega Salesforce, com seu modelo de metadata forte e padrões de CI/CD bem definidos, é um excelente lugar para provar isso.
